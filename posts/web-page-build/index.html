使用hugo搭建网站#

一 前期准备#

1）安装go语言#

从All releases - The Go Programming Language下载.msi安装包，并安装到选定目录下。

在终端输入go version以验证安装成功。

2）安装hugo#

在Releases · gohugoio/hugo下载hugo，然后讲hugo.exe所在目录添加至系统环境变量Path下。

在终端输入hugo version验证。
